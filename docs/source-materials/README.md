## How To Use These Files

These files are source material, not approved product copy. Product-facing claims should still be validated against:

- deterministic Brand Health Record data
- diagnosis rules
- treatment library config
- source-period/caveat policy
- stakeholder-approved synthesis docs

When a source material is used to justify a feature, add a citation in the relevant product or data-design doc and keep the raw source here unchanged.
