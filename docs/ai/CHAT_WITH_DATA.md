# Chat With Data

## Purpose

Users should be able to dialog with the data and visuals, not just read summaries.

## Scope

Chat is scoped to:
- active brand
- active visual
- active category lens
- diagnosis definitions
- treatment definitions
- evidence items
- data caveats

## Required UI behavior

- Suggested questions change based on active visual.
- Chat answers include evidence or missing-data caveat.
- Chat can switch tone between Brand Manager and Insights Lead modes.
- Chat can explain charts, rules, evidence, and treatments.

## Example questions

Brand Health Panel:
- What is the one thing I should notice first?
- What does red mean here?
- Is this a level problem or a momentum problem?

Diagnosis:
- Why did this diagnosis fire?
- What else could explain the data?
- What would change the diagnosis?

Treatment Pharmacy:
- Which treatment is foundational?
- What is the lowest-cost useful step?
- Which treatment is highest risk?
- What should we monitor after treatment?
