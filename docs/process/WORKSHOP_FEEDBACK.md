# Workshop Feedback Log

Use this file to capture stakeholder reactions.

## Template

```text
Feedback:
Who said it:
Audience: Brand Manager / Insights / GN / Commercial / Other
Theme: UI / Data / Diagnosis / Treatment / Chat / Governance / Future
Disposition: V1 / Later / Out of scope
Backlog item:
```

## Themes to watch

- Do users believe the diagnosis?
- Do marketers understand the language?
- Do insights leads trust the evidence?
- Are treatment options specific enough?
- Is chat helpful or distracting?
- Are visuals making the diagnosis easier?
