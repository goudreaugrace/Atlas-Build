# Treatment Library

## Concept

Treatments are the brand-equity “pharmacy.” Each treatment is a structured object that can be added, edited, linked, and ranked.

## Treatment families

### Meaningful treatments

- Clarify proposition
- Refresh relevance
- Check product and experience delivery
- Build emotional connection

### Different treatments

- Sharpen point of difference
- Strengthen distinctive brand assets
- Build functional difference
- Build emotive difference
- Demonstrate category leadership

### Salient treatments

- Build mental availability
- Build occasion memory structures
- Increase reach and continuity
- Improve physical presence

### Pricing / value treatments

- Build premium justification
- Check value delivery and price-pack fit
- Refer to RGM pricing study

### Specialist referrals

- Run specialist research deep dive
- Run creative/asset audit
- Run product/sensory diagnostics
- Run shopper/retail execution review

## Treatment card requirements

Every treatment must include:
- name
- tier
- family
- plain-English description
- best-for diagnoses
- not-for diagnoses
- why it fits
- pros
- cons
- time to impact
- cost
- difficulty
- likelihood of success
- owners
- dependencies
- expected metric movement
- follow-up signals

## Foundation-first rule

When the brand has a foundation deficit or underbuilt equity, recommend foundational treatments before acceleration, premium, or transformation treatments.
