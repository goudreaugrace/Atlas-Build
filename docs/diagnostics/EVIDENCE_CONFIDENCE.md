# Evidence and Confidence

## Evidence types

- BBE metric fact
- Benchmark status
- Momentum signal
- BrandZ typology
- Trend series
- Occasion score
- Growth Navigator signal
- Category lens caveat
- Missing data / known blind spot

## Evidence roles

- Supports diagnosis
- Complicates diagnosis
- Rules out alternative
- Indicates missing data

## Confidence labels

### Directional

Useful as a hypothesis; incomplete evidence.

### Supported

Multiple data points support the read, but some gaps remain.

### Validated

Strong evidence across BBE and adjacent data.

### Confirmed

Reserved for future state with outcome validation.

## Counter-evidence is required

Every diagnosis should attempt to show one thing that complicates or softens the read.

If no counter-evidence exists, say: “No material counter-signal found in the active packet.”
