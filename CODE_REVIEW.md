# CODE_REVIEW.md

Review every change against these questions:

1. Does this preserve the single-brand diagnostic journey?
2. Is any diagnosis or treatment hard-coded that should be config-driven?
3. Does the UI remain simple for brand managers?
4. Can insights leads inspect evidence and rules?
5. Is LLM behavior scoped to the active brand and active visual?
6. Are category, pricing, causality, and portfolio migration guardrails preserved?
7. Can a future Supabase data provider replace JSON without rewriting the UI?
8. Did STATUS.md and BACKLOG.md get updated?
